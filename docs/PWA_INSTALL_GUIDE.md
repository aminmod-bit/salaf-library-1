# Установка Salaf Library как приложение (PWA)

Salaf Library поддерживает установку как Progressive Web App (PWA) на все устройства.

## Android (Chrome / Edge)

1. Откройте `salaflibrary.org` в Chrome или Edge
2. Нажмите кнопку «Установить приложение» на сайте
3. Или: меню браузера → «Установить приложение» / «Добавить на главный экран»
4. Подтвердите установку

Приложение появится на главном экране как полноценное приложение.

## iPhone / iPad (Safari)

1. Откройте `salaflibrary.org` в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой внизу)
3. Выберите «На экран Домой»
4. Нажмите «Добавить»

Приложение появится на главном экране.

## Windows / Mac / Linux (Chrome / Edge)

1. Откройте `salaflibrary.org` в Chrome или Edge
2. В адресной строке нажмите иконку установки (монитор со стрелкой)
3. Или: меню браузера → «Установить Salaf Library»
4. Подтвердите

Приложение откроется в отдельном окне без адресной строки.

## Что доступно офлайн

- Оболочка сайта (навигация, страницы)
- Данные книг (JSON)
- Ранее открытые страницы из кэша

## Что НЕ доступно офлайн

- PDF/EPUB файлы книг (не кэшируются автоматически)
- Аудиофайлы
- Загрузка новых данных

Для доступа к книгам офлайн — сохраняйте их локально через читалку (будущая функция).

## Как обновляется приложение

Приложение обновляется автоматически при открытии, если доступна новая версия.
Вы увидите уведомление «Доступно обновление» — нажмите «Обновить».

## Проверка PWA (Lighthouse)

1. Откройте Chrome DevTools (F12)
2. Перейдите на вкладку Lighthouse
3. Выберите «Progressive Web App»
4. Нажмите «Analyze page load»

Целевые показатели:
- Manifest валиден
- Service worker работает
- Сайт installable
- Нет критических ошибок
