## Главные задачи

1. Полный UI/UX аудит.
2. Убрать слабые места дизайна.
3. Сделать интерфейс единым: карточки, папки, кнопки, цвета, типографика.
4. Довести читалку PDF до уровня Kindle / Apple Books / PDF Expert.
5. Довести импорт книг до максимально автоматического.
6. Сделать админку удобной для долгосрочного управления библиотекой.
7. Подготовить Cloudflare R2/Workers/Pages для будущего.
8. Сделать проект installable PWA и совместимым с Capacitor.
9. Гарантировать производительность на телефоне.
10. Не перегревать устройство: избегать тяжёлого рендера, лишних MutationObserver, массового PDF.js на главной.
