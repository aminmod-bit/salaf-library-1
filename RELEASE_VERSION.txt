Salaf Library — Production UX Release 2.2

Дата сборки: 2026-06-22

Ключевые улучшения:
- добавлена интеграция Quran.com public API для раздела Коран;
- Quran теперь поддерживает Surah / Verse / Juz navigation;
- добавлены маршруты /#/quran/:surahNumber, /#/quran/:surahNumber/:ayahNumber, /#/quran/juz/:juzNumber;
- Quran reader загружает главы, аяты, переводы, ресурсы тафсира и чтецов через API Quran.com;
- добавлен выбор перевода, тафсира и чтеца;
- добавлен запуск аудио суры через recitation API;
- добавлен раздел Хадисы /#/hadith в стиле навигации по сборникам;
- добавлен hadith-books.json со сборниками Isnād и внешними ссылками на первоисточник;
- добавлена навигация Hadith в Sidebar/Header;
- статьи остаются отдельным разделом с языковой фильтрацией и карточным интерфейсом;
- контент сторонних сайтов не копируется без разрешения: используются публичные API/ссылки и собственный интерфейс;
- сохранены PDF Reader, авто-обложки, авто-анализ PDF, языковая система, SEO и GitHub API админка.

Важно:
- Quran.com данные загружаются через их публичный API на стороне клиента.
- Isnād хадисы пока открываются как внешние первоисточники; локальный импорт полного текста хадисов нужно делать только при наличии разрешения/открытого датасета.
- Дизайн разделов вдохновлён логикой навигации указанных сайтов, но не копирует их визуально один-в-один.
