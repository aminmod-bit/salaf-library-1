Salaf Library — Automatic Book Import Release 3.6

Дата сборки: 2026-06-28

Полностью переработана система импорта книг.

Новый процесс:
- пользователь выбирает PDF-файлы пакетно;
- система автоматически читает первые страницы PDF;
- определяет название книги;
- определяет оригинальное арабское название, если оно найдено;
- определяет автора по тексту и известным правилам;
- определяет язык книги;
- определяет количество страниц;
- определяет размер файла;
- извлекает дату создания PDF, если она есть;
- пытается найти ISBN;
- пытается найти издательство;
- пытается найти переводчика;
- пытается найти редактора;
- определяет категорию по названию, тексту, ключевым словам и оглавлению;
- выставляет confidence и статус needsReview;
- создаёт slug;
- создаёт cover.webp и thumb.webp из первой страницы PDF;
- связывает книгу с существующим автором, если автор уже найден в biographies.json;
- если автора нет, создаёт новую запись автора;
- автоматически добавляет PDF в правильную папку Books/{Category};
- создаёт sidecar JSON рядом с PDF;
- обновляет public/data/books.json;
- обновляет public/data/biographies.json;
- создаёт public/data/search-index.json;
- импорт выполняется пакетно через GitHub Git Data API;
- после обработки показывается отчёт: импортировано, авторы найдены, новые авторы, категории автоматически, требуют проверки, обложки созданы.

Поддерживаемые категории:
- Акыда
- Таухид
- Манхадж
- Тафсир
- Хадисы
- Фикх
- Сира
- Азкары
- Дуа
- Биографии
- История
- Арабский язык
- Воспитание
- Даава
- Другое
