Salaf Library — Production UX Release 2.3

Дата сборки: 2026-06-22

Фокус релиза:
- полностью переработана читалка PDF-книг.

Идеи UX взяты из лучших практик Kindle / Apple Books / Google Play Books / Adobe Reader:
- страница по умолчанию подгоняется под экран, а не становится слишком большой;
- отдельный режим «Страница» для комфортного чтения на компьютере;
- режим «По ширине» для планшета и мобильного;
- нижняя панель управления как в приложениях для чтения;
- быстрый слайдер страниц;
- кликабельные зоны слева/справа для переключения страниц;
- поддержка клавиатуры: стрелки, PageUp/PageDown, F, W, Esc;
- свайп на телефоне для перехода страниц;
- фокус-режим без верхней панели;
- полноэкранный режим;
- боковая панель с закладками, поиском и оглавлением;
- поиск по PDF;
- закладки с заметками;
- сохранение прогресса;
- продолжение с последней страницы;
- адаптация под desktop/tablet/mobile.
