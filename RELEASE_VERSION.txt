Salaf Library — Book Languages Folders Release 3.3

Дата сборки: 2026-06-25

Изменение:
- на странице /#/book-languages добавлены папки внутри каждого языкового раздела;
- сначала пользователь видит языки;
- после выбора языка открываются тематические папки книг внутри выбранного языка;
- папки отображаются даже если пока пустые, чтобы структура была готова к наполнению;
- если в папке есть книги, кнопка ведёт в /#/books с фильтром language и category;
- если папка пустая, показывается «Будет добавлено»;
- структура папок соответствует основной библиотеке: Акыда, Таухид, Манхадж, Хадисы, Сира, Фикх, Тафсир, Азкары, Дуа, Арабский язык, Воспитание, История, Биографии, Детские книги, Даава, Другие разделы.

Проверка:
- npm run build: OK
- npm run check: OK
- npm audit moderate+: 0 vulnerabilities
