Salaf Library — Folder Structure Release 3.7

Дата сборки: 2026-06-28

Выполнено:
- все основные разделы теперь открываются через папки/категории, а не сразу через общий список;
- Книги: сначала тематические папки, затем книги выбранной папки;
- Азкары: сначала папки азкаров, затем содержимое папки;
- Хадисы: сначала папки сборников, затем страница выбранного сборника, готовая для глав и хадисов;
- Книги на разных языках: сначала языки, затем папки внутри выбранного языка;
- Статьи: сначала тематические папки, затем статьи выбранной папки;
- Биографии: сначала папки биографий, затем записи выбранной папки;
- добавлен единый компонент FolderCard для профессионального дизайна папок;
- структура масштабируема для большого количества материалов;
- сохранена пустая библиотека, готовая к наполнению;
- сохранён автоматический импорт книг через папки Books/*;
- сохранена PWA, SEO, админка и мультиязычность.
