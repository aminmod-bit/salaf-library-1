Salaf Library — Production UX Release 2.0

Дата сборки: 2026-06-20

Фокус релиза:
- исправление языковой системы по принципу IslamQA: интерфейс выбранного языка должен быть максимально на одном языке;
- контент не подменяется искусственными переводами: если материала на языке нет, показывается честное пустое состояние;
- PDF-книги остаются оригинальными, как и было согласовано.

Ключевые изменения:
- добавлен GlobalTextTranslator: слой автоматического перевода оставшихся hardcoded UI-строк;
- расширены словари UI для EN / AR / TG / UZ / FA;
- Home, Books, Header, Sidebar, Quran и многие общие кнопки теперь меняются глубже при смене языка;
- Fawaid/articles теперь фильтруются по языку: русские фаваиды не показываются как таджикские/узбекские/персидские;
- существующие фаваиды помечены language=Русский;
- добавлена честная логика для материалов: если контента на выбранном языке нет, он не подделывается;
- сохранён раздел Book Languages для наполнения материалов по языкам;
- сохранены Quran placeholders и сообщения о недостающих переводах;
- сохранены PDF auto covers, PDF.js Reader, аудиоплеер, SEO-каталог и GitHub API админка.

Проверка:
- npm run build: OK
- npm run check: OK
- npm audit moderate+: 0 vulnerabilities

Дальше:

- добавлять реальные материалы на таджикском, узбекском, персидском, арабском и английском;
- после добавления Telegram-ссылки встроить её в Header / Sidebar / About / Report.
