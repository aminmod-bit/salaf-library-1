Salaf Library — Production UX Release 1.6

Дата сборки: 2026-06-20

Ключевые улучшения:
- автоматические обложки из 1-й страницы PDF теперь кэшируются в IndexedDB;
- при повторном открытии каталога обложки не рендерятся заново, а берутся из локального кэша браузера;
- сохранён fallback на премиальную сгенерированную обложку, если PDF недоступен;
- админский импорт теперь асинхронно читает первые страницы загружаемого PDF;
- при импорте автоматически улучшаются название, автор, категория и теги на основе текста PDF;
- добавлен общий модуль src/utils/pdfMetadata.ts для анализа текста PDF;
- сохранены языки RU/EN/AR/TG/UZ/FA;
- сохранён раздел Коран;
- сохранён PDF.js Reader, закладки, заметки, поиск по PDF, оглавление и прогресс чтения;
- сохранён SEO-каталог и production sitemap.
