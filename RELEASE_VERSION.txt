Salaf Library — Production UX Release 1.5

Дата сборки: 2026-06-20

Ключевое улучшение:
- автоматические обложки книг теперь пытаются рендерить 1-ю страницу PDF прямо в браузере;
- если PDF доступен и CORS позволяет, карточка книги автоматически показывает первую страницу PDF как обложку;
- если PDF не удалось отрендерить, остаётся премиальная сгенерированная обложка с названием, автором и категорией;
- рендер обложек ленивый: запускается только когда карточка появляется рядом с экраном;
- PDF.js загружается динамически, чтобы не ломать скорость главной страницы.

Сохранено из прошлых релизов:
- PDF.js Reader;
- поиск по PDF;
- закладки и заметки;
- оглавление PDF;
- прогресс чтения;
- улучшенный аудиоплеер;
- языки RU/EN/AR/TG/UZ/FA;
- раздел Коран;
- SEO-каталог;
- официальный каталог 116 PDF-книг.

- npm run build: OK
