Salaf Library — Production UX Release 1.8

Дата сборки: 2026-06-20

Ключевые улучшения:
- расширена языковая система: ключевые интерфейсные блоки Home, Books, Header, Sidebar и Quran реагируют на смену языка;
- добавлены расширенные словари UI для RU/EN/AR/TG/UZ/FA;
- исправлено поведение LanguageSwitcher и supportedLngs;
- добавлены языковые названия для будущего раздела книг по языкам;
- раздел Коран расширен до списка всех 114 сур;
- добавлена поддержка переводов аятов по текущему языку;
- добавлена поддержка тафсира в карточке аята;
- добавлены placeholder-состояния для сур, текст которых ещё не загружен;
- сохранены авто-обложки из первой страницы PDF с IndexedDB-кэшем;
- сохранён авто-анализ PDF для автора, названия, категории и тегов;
- сохранены PDF Reader, аудиоплеер, SEO-каталог и GitHub API админка.

Проверка:
- npm run build: OK
- npm run check: OK
- npm audit moderate+: 0 vulnerabilities

Следующий этап:
- отдельная страница книг по языкам;
- перевод оставшихся страниц и административных экранов;
